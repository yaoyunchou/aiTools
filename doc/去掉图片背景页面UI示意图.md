# 去掉图片背景页面 UI 示意图

---

```
┌───────────────────────────────┐
│        图片去背景            │
├───────────────────────────────┤
│   [ 上传图片按钮 ]            │
├───────────────────────────────┤
│   [ 确认处理按钮 ]            │
├───────────────────────────────┤
│   处理结果：                  │
│   ┌─────────────┐             │
│   │  处理后图片 │             │
│   └─────────────┘             │
├───────────────────────────────┤
│   [ 是否发布到广场：开关 ]     │
└───────────────────────────────┘
```

---

## 说明
- 顶部为页面标题"图片去背景"
- 上传图片按钮，选择本地图片
- 确认处理按钮，点击后进行AI去背景
- 处理结果区，展示去背景后的图片
- 是否发布到广场，使用开关或勾选框
