# 照片换脸页面 UI 示意图

---

```
┌───────────────────────────────┐
│         照片换脸              │
├───────────────────────────────┤
│   [ 上传图片按钮 ]             │
├───────────────────────────────┤
│   [ 确认处理按钮 ]            │
├───────────────────────────────┤
│   处理结果：                  │
│   ┌─────────────┐             │
│   │  处理后图片 │             │
│   └─────────────┘             │
├───────────────────────────────┤
│   [ 是否发布到广场：开关 ]     │
└───────────────────────────────┘
```

---

## 说明
- 顶部为页面标题"照片换脸"
- 上传图片按钮，选择本地图片
- 确认处理按钮，点击后进行AI换脸
- 处理结果区，展示换脸后的图片
- 是否发布到广场，使用开关或勾选框
